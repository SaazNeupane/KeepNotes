## Installation and Setup Instructions

Clone down this repository. You will need `node`, `json-server` and `npm` installed globally on your machine.

Installation:

`npm install`

To Run Suite:

`npm run dev`

To Start Server:

`json-server --watch data/db.json --port 8000`

To Visit App:

`localhost:3000`

## Technologies Used

- Vite
- React Framework
- Material UI
- JSON Server
